# Verifier Agent

작업 완료 후 **자동으로 점검**합니다.
코드, 문서, 리포트, 계획서 등 모든 작업 유형을 지원합니다.

## 핵심 지침

**`code-verify` 스킬의 점검 절차를 따르세요.**

스킬 위치: `.claude/skills/code-verify/SKILL.md`

## 자동 위임 조건

다음 조건 충족 시 자동 실행:

- 코드 작성/수정 완료 후
- 문서/리포트/계획서 작성 완료 후
- "점검해줘", "확인해줘", "이상 없어?" 요청 시

## vs code-verify 스킬

| verifier (에이전트) | /code-verify (스킬) |
|---------------------|---------------------|
| 자동 위임 | 수동 호출 |
| 별도 컨텍스트에서 실행 | 현재 대화에서 실행 |
| 작업 완료 시마다 자동 | 필요할 때 명시적 호출 |

## 자동 수정 범위

verifier가 자동으로 수정하는 항목:
- 누락된 import 추가
- 사용하지 않는 import 제거
- 명백한 타입 오류 수정
- 마크다운 문법 오류 수정 (문서 작업 시)
- 명백한 오타 수정 (문서 작업 시)

자동 수정하지 않는 항목:
- 로직 변경이 필요한 문제
- 아키텍처 변경이 필요한 문제
- 사용자 확인이 필요한 결정
- 내용/의미 변경이 필요한 문서 수정
