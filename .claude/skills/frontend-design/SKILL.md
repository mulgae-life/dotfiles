이 스킬은 "AI 느낌 나는" 뻔한 디자인을 피하고, 독창적이며 프로덕션 수준의 프론트엔드 인터페이스를 만드는 가이드입니다. 미적 디테일과 창의적 선택에 세심한 주의를 기울여 실제 작동하는 코드를 구현합니다.

사용자가 제공하는 프론트엔드 요구사항: 컴포넌트, 페이지, 애플리케이션, 인터페이스 등. 목적, 대상 사용자, 기술적 제약사항 등의 맥락이 포함될 수 있습니다.

## 디자인 사고

코딩 전에 맥락을 이해하고 **대담한** 미학적 방향을 설정하세요:
- **목적**: 이 인터페이스가 해결하는 문제는? 누가 사용하는가?
- **톤**: 극단적인 방향을 선택하세요: 극도로 미니멀, 맥시멀리스트 혼돈, 레트로-퓨처리스틱, 유기적/자연적, 럭셔리/세련됨, 장난스러움/장난감 같은, 에디토리얼/매거진, 브루탈리스트/날것, 아르데코/기하학적, 소프트/파스텔, 인더스트리얼/실용적 등. 선택지는 무궁무진합니다. 영감을 얻되 미학적 방향에 충실한 디자인을 하세요.
- **제약사항**: 기술적 요구사항 (프레임워크, 성능, 접근성).
- **차별화**: 무엇이 이것을 **잊을 수 없게** 만드는가? 사람들이 기억할 단 하나는?

**핵심**: 명확한 컨셉 방향을 선택하고 정밀하게 실행하세요. 대담한 맥시멀리즘과 세련된 미니멀리즘 모두 효과적입니다 - 중요한 것은 강도가 아니라 의도성입니다.

그 다음 작동하는 코드 (HTML/CSS/JS, React, Vue 등)를 구현합니다:
- 프로덕션 수준이며 기능적
- 시각적으로 인상적이고 기억에 남음
- 명확한 미학적 관점으로 일관성 있음
- 모든 디테일이 꼼꼼하게 다듬어짐

## 프론트엔드 미학 가이드라인

집중해야 할 요소:
- **타이포그래피**: 아름답고 독특하며 흥미로운 폰트를 선택하세요. Arial, Inter 같은 뻔한 폰트를 피하고 프론트엔드의 미학을 높이는 독특한 선택을 하세요. 예상치 못한 개성 있는 폰트 조합. 독특한 디스플레이 폰트와 세련된 본문 폰트를 조합하세요.
- **컬러 & 테마**: 일관된 미학을 유지하세요. CSS 변수를 사용해 일관성을 확보하세요. 날카로운 악센트가 있는 지배적 컬러가 소심하고 균등하게 분배된 팔레트보다 효과적입니다.
- **모션**: 효과와 마이크로 인터랙션에 애니메이션을 사용하세요. HTML에서는 CSS 전용 솔루션을 우선하세요. React에서는 가능하면 Motion 라이브러리를 사용하세요. 임팩트 있는 순간에 집중하세요: 엇갈린 등장 효과(animation-delay)가 있는 잘 조율된 페이지 로드 하나가 산발적인 마이크로 인터랙션보다 더 큰 즐거움을 줍니다. 놀라움을 주는 스크롤 트리거와 호버 상태를 사용하세요.
- **공간 구성**: 예상치 못한 레이아웃. 비대칭. 겹침. 대각선 흐름. 그리드를 깨는 요소. 넉넉한 여백 또는 통제된 밀도.
- **배경 & 시각적 디테일**: 단색 기본값 대신 분위기와 깊이를 만드세요. 전체 미학에 맞는 맥락적 효과와 텍스처를 추가하세요. 그라데이션 메시, 노이즈 텍스처, 기하학적 패턴, 레이어드 투명도, 드라마틱한 그림자, 장식적 테두리, 커스텀 커서, 그레인 오버레이 같은 창의적 형태를 적용하세요.

뻔한 AI 생성 미학은 **절대 사용하지 마세요**: 과도하게 사용되는 폰트 패밀리(Inter, Roboto, Arial, 시스템 폰트), 진부한 컬러 스킴(특히 흰 배경에 보라색 그라데이션), 예측 가능한 레이아웃과 컴포넌트 패턴, 맥락에 맞는 캐릭터가 없는 판에 박힌 디자인.

창의적으로 해석하고 맥락에 맞게 진정으로 디자인된 느낌의 예상치 못한 선택을 하세요. 같은 디자인은 없어야 합니다. 라이트와 다크 테마, 다른 폰트, 다른 미학을 다양하게 사용하세요. 여러 생성에 걸쳐 흔한 선택(예: Space Grotesk)으로 수렴하지 **절대 마세요**.

**중요**: 구현 복잡도를 미학적 비전에 맞추세요. 맥시멀리스트 디자인은 광범위한 애니메이션과 효과가 있는 정교한 코드가 필요합니다. 미니멀리스트나 세련된 디자인은 절제, 정밀함, 간격·타이포그래피·섬세한 디테일에 대한 세심한 주의가 필요합니다. 우아함은 비전을 잘 실행하는 데서 나옵니다.

기억하세요: Claude는 놀라운 창의적 작업이 가능합니다. 주저하지 말고, 틀을 벗어나 생각하고 독특한 비전에 완전히 몰입할 때 진정으로 창조할 수 있는 것을 보여주세요.
