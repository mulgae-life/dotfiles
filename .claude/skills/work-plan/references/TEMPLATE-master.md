# Master 템플릿 (Split 모드)

Split 모드의 최상위 계획서. **전체 맥락·Phase 맵·리스크·검증**을 담당. Phase별 상세는 part 파일로 분리.

> master는 작업 내내 컨텍스트에 상주한다. 간결하게 유지한다.
> 섹션 헤더 `(필수)`/`(선택)` 태그 준수. `(선택)` 섹션은 해당 없으면 섹션 자체를 생략.
